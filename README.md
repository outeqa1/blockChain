# Blockchain
This project shows how blockchain works by the example of a lender-borrower relationship.
Realized on flask framework.

## Installation
```
git clone https://github.com/outeqa1/blockChain/
```
pip install -r requirements.txt


## Run app
```
python main.py

Go to http://127.0.0.1:5000/
```
## How it works
```
Fill in the form as you wish 
```
Press 'Submit'
```
If you want to check integrity press 'Check integrity' button
